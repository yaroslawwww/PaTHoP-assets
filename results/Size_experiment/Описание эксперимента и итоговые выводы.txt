Данный эксперимент сравнивает 3 метрики при предсказании на 10 точек вперед:RMSE,количество непрогнозируемых точек,MAPE для разных обучающих выборок.
Были посчитаны метрики для обучающей выборки,состоящей из 10000 точек ряда с  σ=10, r=28, b=8/3,для
которого мы предсказываем и переменным количеством точек ряда с другим значением r.
Были подсчитаны метрики для обучающей выборки из переменного количества точек базового ряда с  σ=10, r=28, b=8/3(на графике обозначено как r=28).
На графике изображена зависимость метрик от суммарного размера обучающей выборки.
В результате эксперимента получено,что для небольших отклонений на разных участках может наблюдаться улучшение всех трех метрик,сравнимое с добавлением выборки оригинального ряда.Для добавленного ряда с r=27.9 и r=27.99 получаются результаты сравнимые по всем метрикам с выборкой только на основе базового ряда.При этом для r=28.0001 результаты получаются хуже чем для r=28.1.Все метрики уменьшают количество непрогнозируемых точек.

Также в файле unlucky_experiments.png можно увидеть  менее удачные расширения обучающих выборок за счет добавления выборок с рядами с параметрами r=28.01(почему предсказание хуже чем с r=28.1,r=28.0001 меня вгоняет в тупик),r=30,r=38,r=27,поскольку они не дают выигрыш по всем трем метрикам,но могут дать выигрыш по количеству непрогнозируемых точек или метрикам качества предсказания.
